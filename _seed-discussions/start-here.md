# 👋 Start here — how to get help with Label Alchemy

Welcome to the Label Alchemy community! This is the public hub for support, feedback, and the
roadmap. **Label Alchemy** is the VS Code extension that converts user-facing hard-coded strings
in Salesforce Apex, LWC, and Aura code into Custom Labels.

> The extension's source code is **not** hosted here. This space is for bug reports, feature
> requests, questions, announcements, and the public roadmap.

## Before you post

1. **Search** [existing issues](https://github.com/Ramprasad94/labelalchemy/issues?q=is%3Aissue) and [discussions](https://github.com/Ramprasad94/labelalchemy/discussions) — your question may already be answered.
2. Check the **[docs & FAQ](https://docs.labelalchemy.dev)**.
3. For bugs, include your extension version, VS Code version, OS, and a small **redacted** repro snippet.

## ⚠️ Please never share secrets

Issues and discussions here are **public**. Never paste API keys, license keys, org credentials, or
proprietary source code. A small redacted snippet is all that's needed to reproduce most bugs. For
anything sensitive, email **support@labelalchemy.dev** instead.

## Response expectations

Label Alchemy is built and maintained by a solo developer. Every issue and discussion is read, and
I respond as quickly as I reasonably can — typically within a few business days. Paid customers who
email **support@labelalchemy.dev** get priority. There's no formal SLA, but I genuinely care about
getting you unblocked.
